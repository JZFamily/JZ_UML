# JZ_UML
UML tool for jzf

# 需求分析

平时写代码的时候，在固定套路下去创建文件，编写类，实现接口，真的贼无趣。在上次的proto到local_class转换中，我感觉做一个代码生成工具帮助做一些重复工作还是不错，但是过程抽象和代码编写一直无法做到通用化。

这次看到Bouml工具，本以为会搞到一个比较不错的东西，但是他的界面实在是丑，可用性实在是差。觉得还是按照自己的思路来做一个比较好。

## 暂停

* uml到c++
